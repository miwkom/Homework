Библиотека requests позволяет python'у работать со ссылками.
Например, функция get позволяет pyhton'у подключаться к HTTP.
Функция headers дает заголовки сайта на момент подключения. Заголовки содержат различную информацию например: информация о сервере, дата, кодировка.
Функция content возвращает содержимое в байтах.
Таким образом, с использованием функцией with можно сделать запрос на картинку и создать файл на его основе.

Библиотека pillow дает python работать с изображениями.
